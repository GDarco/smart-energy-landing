# Інструкція з генерації Javadoc

Для створення технічної документації проєкту використовується стандартний утиліта `javadoc`.

## Команда для генерації:
Виконайте в корені проєкту:
`javadoc -d docs/api -sourcepath src -subpackages .`

## Результат:
Документація буде доступна у папці `docs/api/index.html`.
